# 任务看板

任务看板是 FylloCode 的任务入口。它负责把本地任务和已接入研发系统中的任务集中起来，让用户从具体任务发起 Agent 讨论。

<figure class="fc-doc-image">
  <img src="/assets/screenshots/task.png" alt="任务看板截图" />
</figure>

## 主要能力

- 查看打开和关闭状态的任务
- 创建本地任务
- 从任务卡片发起讨论
- 为后续 Proposal 提供任务背景
- 为云效等外部任务系统保留接入位置

## 使用建议

Task 不应该只是一个标题。为了让后续 Proposal 可审查，建议在任务描述中包含：

- 问题背景
- 当前行为和期望行为
- 影响范围
- 约束条件
- 验收标准

当任务信息不足时，应该先在对话中澄清，而不是直接进入实现。
