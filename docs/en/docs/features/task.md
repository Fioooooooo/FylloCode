# Task Board

Task Board is the task entry point in FylloCode. It gathers local tasks and tasks from connected engineering systems so users can start Agent discussions from concrete work items.

<figure class="fc-doc-image">
  <img src="/assets/screenshots/task.png" alt="Task board screenshot" />
</figure>

## Main Capabilities

- View open and closed tasks
- Create local tasks
- Start a discussion from a task card
- Provide task context for later Proposals
- Reserve integration points for external task systems such as Yunxiao

## Usage Advice

A Task should be more than a title. To make the later Proposal reviewable, include:

- Problem context
- Current behavior and expected behavior
- Impact scope
- Constraints
- Acceptance criteria

When task information is insufficient, clarify it in Chat before moving into implementation.
