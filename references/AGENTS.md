# References

当前目录中的文件为针对第三方的调研，以及 FylloCode 自身的较大改动方案。

第三方调研结果可能会随时间过期；FylloCode 的改动方案也可能随功能演进逐渐失效。

所以这个目录中的所有文档 **仅供参考，不是事实依据**。

## 目录结构

```
references/
├── designs/              # FylloCode 自身设计方案
│   ├── bundled-mcp-http/ # Bundled MCP Server HTTP 化设计
│   ├── domain-refactor/  # 领域重构参考设计
│   ├── fyllo-action/     # Fyllo Action 架构整改方案
│   ├── fyllo-spawn/      # fyllo-spawn MCP Server 设计
│   └── knowledge-tool/   # Knowledge Tool 参考设计
└── third-party/          # 第三方协议、平台调研
    ├── acp/              # ACP 协议相关调研
    └── integrations/     # 第三方平台集成（云效等）
```
