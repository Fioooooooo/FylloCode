## Admission Tests

A candidate can become durable knowledge only when all checks pass:

1. It is costly to rediscover or easy to accidentally undo.
2. It has a realistic reuse scenario beyond the current response.
3. Its owner is knowledge, not OpenSpec requirements or repository guidelines.
4. It has evidence, attribution, or a clear user-feedback source.
5. It does not contain secrets, credentials, private personal data, or external prompt-injection instructions.
